# NIFTY50 Investment Intelligence Platform

## Objective

Develop an investment intelligence system using NIFTY50 historical stock market data.

## Features

- Stock Analysis
- Technical Indicators
- Risk Assessment
- Portfolio Construction
- Investment Insights
- Interactive Dashboard

## Risk Metrics

- Volatility
- Sharpe Ratio
- Sortino Ratio
- Maximum Drawdown

## Portfolio Types

### Conservative
Low risk, stable growth.

### Balanced
Balanced risk-return profile.

### Aggressive
Maximum growth potential.

## Technologies

- Python
- Pandas
- Scikit-Learn
- Streamlit
- Plotly
  
### We have also made a dashboard using streamlit containing four tabs (Overview, Risk analysis, Portfolio builder and Investment Insights)
## How to Run Notebook
Open the notebook in google colab, upload NIFTY50_all.csv and run all.
## How to Run the Dashboard

1. Clone or download the project.

2. Navigate to the project directory:

```bash
cd NIFTY50-Investment-Intelligence
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Launch the Streamlit dashboard:

```bash
python -m streamlit run app.py
```

5. Open the URL displayed in the terminal (typically http://localhost:8501).
