# NIFTY50 Investment Intelligence Platform

## Objective

Develop an investment intelligence system using NIFTY50 historical stock market data.

## Features

- Stock Analysis
- Technical Indicators
- Risk Assessment
- Portfolio Construction
- Investment Insights
- Interactive Dashboard

## Risk Metrics

- Volatility
- Sharpe Ratio
- Sortino Ratio
- Maximum Drawdown

## Portfolio Types

### Conservative
Low risk, stable growth.

### Balanced
Balanced risk-return profile.

### Aggressive
Maximum growth potential.

## Technologies

- Python
- Pandas
- Scikit-Learn
- Streamlit
- Plotly

## How to Run

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```
